Матрицы в заданиях двумерные, если не указано иное.
1. Написать функции ввода (случайного заполнения) и вывода матрицы.
2. Написать функцию транспонирования квадратной матрицы A[n][n] (без использования второй матрицы).
3. Написать функцию перемножения матриц A[n][m] и B[m][r].
4. Написать функцию копирования подстроки длины k, начиная с i-го символа.
5. В целочисленном массиве A найти число, повторяющееся максимальное количество раз.
6. Задан массив целых чисел размерности n. Найти самую длинную подпоследовательность подряд идущих нулей. Вывести начало и длину найденной подпоследовательности.
7*. Написать программу приведения квадратной матрицы к треугольному виду.
8*. Написать программу вычисления определителя матрицы.
9*. Шифр Цезаря (или другой). Написать функции шифрования и расшифрования.
